volatile uint16_t drv_BoostConverter_Initialize ( volatile struct BOOST_CONVERTER_s *  boostInstance)

This function initializes all peripheral modules and their instances used by the power controller.

Parameters
boostInstancePointer to a Boost Converter data object of type struct BOOST_CONVERTER_s
Returns
unsigned integer (0=failure, 1=success)

This function initializes the PWM module, PWM channels, ADC channels for temperature, input voltage, output voltage and phase current. The boost IO pins are also initialize while keeping the boost converter operation disabled. The state machine is set to Initialize.

◆ drv_BoostConverter_Execute()

volatile uint16_t drv_BoostConverter_Execute ( volatile struct BOOST_CONVERTER_s *  boostInstance)

This function is the main boost converter state machine executing the most recent state.

Parameters
boostInstancePointer to a Boost Converter data object of type struct BOOST_CONVERTER_s
Returns
0 = BOOST_OPSRET_REPEAT
1 = BOOST_OPSRET_COMPLETE
2 = BOOST_OPSRET_REPEAT

This function performs tasks in the state machine.

  • If state machine state returns ERROR, switch to ERROR state in next execution cycle
  • If state machine state signals state completion, move on to next state in line
  • When state machine state returns REPEAT, the recent state function will be called again
  • When state machine state returns an unknown result, the state machine will be reset to INITIALIZE again
